# Описание модульной структуры

1. **src/main.py**: Основной файл для запуска программы.
2. **src/io_handler.py**: Модуль для ввода данных пользователем и вывода чека.
3. **src/calculator.py**: Модуль с формулами расчета стоимости.
4. **src/discounts.py**: Модуль для расчета скидок и наценок.
5. **src/utils.py**: Валидация ввода и вспомогательные функции.
