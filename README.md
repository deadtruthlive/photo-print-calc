# Индивидуальный проект по УП.02: Калькулятор стоимости печати фотографий

## 1. Назначение программы
Программа представляет собой консольное приложение для автоматизации работы фотосалона. Она позволяет пользователю рассчитать итоговую стоимость заказа на печать фотографий с учетом выбранных параметров, объема заказа и наценки за срочность.

## 2. Входные данные
В процессе работы программа запрашивает у пользователя следующие данные:
*   **Формат фотографии:** выбор из предложенных вариантов (10x15, 13x18, A4, A3).
*   **Тип бумаги:** матовая или глянцевая.
*   **Количество фотографий:** целое положительное число.
*   **Срочность исполнения:** логический выбор (да/нет).

## 3. Выходные данные
Результатом работы программы является сформированный «чек», содержащий:
*   Подтверждение выбранных параметров (формат, бумага, количество).
*   Базовую цену за одну фотографию.
*   Размер начисленной скидки (при наличии) или наценки.
*   Итоговую стоимость всего заказа.

## 4. Функциональные требования
Программа должна реализовывать следующую логику:
*   **Расчет базовой стоимости:** цена зависит от формата (например, 10x15 — 15 руб., A4 — 50 руб.).
*   **Наценка за тип бумаги:** глянцевая бумага увеличивает стоимость единицы товара на 10%.
*   **Система оптовых скидок:** 
    *   от 50 до 100 фото — скидка 5%;
    *   свыше 100 фото — скидка 10%.
*   **Наценка за срочность:** при выборе срочного заказа общая сумма увеличивается на 25%.
*   **Валидация:** проверка корректности ввода количества (обработка исключений при вводе текста вместо цифр).

## 5. Технологический стек
*   **Язык программирования:** Python 3.8+
*   **Библиотеки:** Стандартная библиотека Python.
